## Basic Info

This game was created for **HackYeah**.

It was developed in **2023 during a 24-hour hackathon**.

## Game Info

It is a **tycoon-like 2D game** focused on upgrading dinosaurs and using them to fight.

It is one of the **worst games released by our team**, but the project helped us improve and gain valuable experience.

The game can be played here:
https://steellotus.itch.io/dino-reevolve

## Technology

The game was developed using **Unity** with the **C# programming language**.
